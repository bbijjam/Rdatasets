coffee_importsR Documentation

Coffee Imports for Select Importing Countries

Description

A simple time series on coffee imports for select importing countries (i.e. European Union + Japan + Russia + Tunisia + United States).

Usage

coffee_imports

Format

A data frame with 30 observations on the following 3 variables.

year

the year

imports

coffee imports for all select importing countries (in thousand 60-kg bags)

usaimports

coffee imports for just the United States (in thousand 60-kg bags)

Details

Data come from the International Coffee Organization, of which I feel I should be a member.
